##Deno 2 REST API


deno installation

Install the Deno runtime on your system using one of the terminal commands below.

macOS:
curl -fsSL https://deno.land/install.sh | sh

Windows:
irm https://deno.land/install.ps1 | iex

Linux:
curl -fsSL https://deno.land/install.sh | sh


You can verify the installation by running:
deno --version
